Cosmic Survivor

Cyber Roguelike Shooter
CONCEPT / コンセプト
「無限に広がるサイバー宇宙を舞台にした、スリリングな全方位ローグライク・サバイバルシューティング」
迫り来る無数の敵を駆逐し、経験値を回収して自身を強化。3分間生き残り、深宇宙の支配者である最終ボスを撃破せよ！
KEY FEATURES / 工夫ポイント
  • 全方位スクロール＆カメラ追従: プレイヤーを中心にカメラが滑らかに動き、広大なフィールドを縦横無尽に飛び回れる開放的なプレイ感。
  • キー・マウス・スワイプのマルチ操作: WASD/矢印キーに加え、マウスドラッグやスマホのタッチ移動に対応。直感的で快適な全方位移動を実現。
  • Web Audio APIによる動的効果音合成: 外部音声ファイルはゼロ。波形合成により、レトロ感溢れるビーム音、爆発音、回復チャイムなどをプログラムのみで生成。
  • ピンチ救済ドロップメカニズム: プレイヤーのHPが40%以下になると、敵撃破時の回復アイテム（➕）のドロップ率が最大12%まで上昇する動的な難易度調整を搭載。
  • 美麗なサイバーエフェクト: 画面シェイク、被弾時の白光フラッシュ、ポップアップダメージテキスト、リジェネ回復時の緑色上昇粒子など、極上のヒット感とフィードバック。
  • 9種の強化スキルビルド: メインビームの多方向貫通、回転オーブ（バリア）、ランダム落雷、自動誘導ドローンなど、自分だけのビルドを構築可能。
